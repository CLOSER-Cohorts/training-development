## What variable metadata should you create?

Thinking back to the type of information we would need to understand the example dataset, what metadata elements could we capture to provide information about the variables? The elements marked with (R) is considered required metadata that all projects should complete. The elements marked with (O) are optional metadata that can be completed if relevant.

**What do the abbreviations mean?**
<details>
<summary>Metadata elements</summary>
<p></p>
  
  - Variable name (R)
  - Variable label (R)
  - Variable description (O)
    
</details>


**What type of data did you collect for the variable?**
<details>
<summary>Metadata elements</summary>
<p></p>
  
  - Value representation (R)
  - Data type (R)
  - Codelist reference (O)
  - Valid range (R)
    
</details>



<p></p>

**How was the variable measured?**
<details>
<summary>Metadata elements</summary>
<p></p>
  
  - Unit of measurement (O)
  - Derived Y/N (R)
  - Data provenance (R)
  - Measurement provenance (R)
  - Source reference (O)
    
</details>






**What was the unit of study?**
<details>
<summary>Metadata elements</summary>
<p></p>
  
  - Unit type (R)
  - Population (R)
  - Universe (R)
    
</details>





**How much data was collected for each data?**
<details>
<summary>Metadata elements</summary>
<p></p>
  
  - Number of cases (R)
  - Missing value code (O)
  - Valid cases (R)
  - Invalid cases (R)
    
</details>
